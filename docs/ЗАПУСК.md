# 🚀 Запуск и сборка — Склад Учёт

## 📱 Быстрый старт (разработка)

### Требования
- **Node.js** 18+
- **Expo Go** на телефоне ([Android](https://play.google.com/store/apps/details?id=host.exp.exponent) / [iOS](https://apps.apple.com/app/expo-go/id982107779))

### Запуск

```bash
cd ~/warehouse-app

# 1. Установить зависимости (один раз)
npm install

# 2. Запустить dev server
npx expo start
```

На телефоне открыть **Expo Go** → Scan QR-код из терминала.

> 📡 Если телефон не в той же Wi-Fi сети, добавьте флаг `--tunnel`:
> ```bash
> npx expo start --tunnel
> ```
> Это tunneliрует через ngrok — работает из любой сети.

## 📦 Сборка APK (Android)

### Способ 1: EAS Build (облачная сборка, рекомендуется)

Без Android Studio. Expo собирает в облаке.

```bash
# 1. Установить EAS CLI
npm install -g eas-cli

# 2. Войти в аккаунт Expo
eas login

# 3. Настроить сборку (создаёт eas.json)
eas build:configure

# 4. Собрать APK для Android
eas build --platform android --profile preview

# 5. (или) AAB для Google Play
eas build --platform android --profile production
```

APK/AAB скачается с EAS — можно установить на телефон.

### Способ 2: Android Studio (локальная сборка)

Требует **Android Studio** и ~20 ГБ свободного места.

```bash
# 1. Собрать native проект
npx expo prebuild --platform android

# 2. Собрать APK
cd android
./gradlew assembleRelease

# 3. APK лежит в:
# android/app/build/outputs/apk/release/app-release.apk
```

### Что установить для Android Studio

| Компонент | Где взять |
|-----------|-----------|
| Android Studio | [developer.android.com/studio](https://developer.android.com/studio) |
| Android SDK (API 35) | SDK Manager → Android 15.0 |
| NDK (Side-by-side) | SDK Manager → SDK Tools → NDK |
| JDK 17+ | `sudo pacman -S jdk17-openjdk` |

После установки SDK, создать файл `~/.bashrc` или `~/.zshrc`:

```bash
export ANDROID_HOME=~/Android/Sdk
export PATH=$PATH:$ANDROID_HOME/emulator
export PATH=$PATH:$ANDROID_HOME/platform-tools
```

## 🍎 Сборка IPA (iOS / iPhone)

### Способ 1: EAS Build (облачная сборка)

Без Xcode и Mac. Expo собирает на своих Mac-mini в облаке.

```bash
# Настройка (один раз)
eas build:configure

# Собрать IPA
eas build --platform ios --profile preview

# Для публикации в App Store
eas build --platform ios --profile production
```

> **Важно:** для EAS iOS сборки нужен **Apple Developer Account** ($99/год)
> и созданные сертификаты/профили в [App Store Connect](https://appstoreconnect.apple.com/).
> EAS Build управляет сертификатами автоматически:
> ```bash
> eas credentials --platform ios
> ```

### Способ 2: Xcode (локальная сборка)

Требует **macOS** и **Xcode** (~30 ГБ).

```bash
# 1. Собрать native проект
npx expo prebuild --platform ios

# 2. Открыть в Xcode
open ios/warehouseapp.xcworkspace

# 3. В Xcode: Product → Archive → Distribute App
```

### Способ 3: Expo Application Service (сборка IPA из .app)

```bash
# Собрать .app архив
npx expo run:ios --configuration Release

# Подписать и упаковать
xcodebuild -exportArchive ...
```

## 🐛 Типичные проблемы

**`npx expo start` — не открывается QR**
> Проверь: телефон в той же Wi-Fi сети? Если нет, используй `--tunnel`.

**Сборка падает на Android — "SDK not found"**
> Проверь: `echo $ANDROID_HOME`. SDK должен быть установлен.

**iOS сборка — "No Apple Developer Account"**
> Нужен Apple Developer Program ($99/год). Без него можно только в симулятор.

**EAS Build — "Build failed"**
> ```bash
> eas build:list          # посмотреть логи
> eas build --platform android --clear-cache  # пересобрать с чистого
> ```
